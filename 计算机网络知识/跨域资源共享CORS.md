# crossOrigin

https://developer.mozilla.org/zh-CN/docs/Web/HTML/Attributes/crossorigin

HTML5的一些 HTML 元素提供对 CORS 的支持， 例如 \<audio>、\<img>、\<link>、\<script> 和 \<video> 有跨域属性 (crossOrigin property)，用来允许元素获取数据的 CORS 请求。
